# csglobal_CSC500-1-milestone-6
This is the sixth Critical assignment for the class PRINCIPLES OF PROGRAMMING of CSU global 

Shopping Cart Program
Description
This program simulates a shopping cart system where users can add, remove, and modify items. It also allows viewing item descriptions and calculating the total cost.

How It Works
ItemToPurchase class: Represents an item with name, price, quantity, and description.
ShoppingCart class: Manages a list of items and provides operations like add, remove, modify, and display.
Features
Add item to cart
Remove item from cart
Change item quantity
Output item descriptions
Output shopping cart with total cost


How to Run
Save the code in a file named shopping_cart.py.
